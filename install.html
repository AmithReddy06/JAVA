Docker Installation Procedure

This procedure outlines the steps to install Docker on your system.

Prerequisites

Before you begin, ensure that your system meets the following requirements:

  • Operating System: Linux (Ubuntu, Debian, CentOS), macOS, or Windows
  • Internet connectivity to download Docker

Installation Steps

Follow the steps below to install Docker:

  1. Linux: Open a terminal window. macOS: Open the Terminal app. Windows: Open PowerShell or Command Prompt.

  2. Update the package index and install necessary dependencies by running the following command:

sudo apt update sudo apt install apt-transport-https ca-certificates curl software-properties-common

  1. Add the Docker GPG key to ensure the authenticity of the Docker repository. Run the following command:

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg –dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

  1. Add the Docker repository to your system’s sources list:

  • For Ubuntu:

    echo "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

  • For Debian:

    echo "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/debian $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

  • For CentOS:

    sudo tee /etc/yum.repos.d/docker.repo <<EOF
[docker-ce-stable]
name=Docker CE Stable - $basearch
baseurl=https://download.docker.com/linux/centos/7/x86_64/stable
enabled=1
gpgcheck=1
gpgkey=https://download.docker.com/linux/centos/gpg
EOF
  1. Update the package index again:

sudo apt update

  1. Install Docker by running the following command:

sudo apt install docker-ce docker-ce-cli containerd.io

  1. Verify that Docker is installed correctly by running the following command to check the Docker version:

docker –version

You should see the version number of Docker if the installation was successful.

  1. (Optional) To run Docker commands without sudo, add your user to the “docker” group:

sudo usermod -aG docker $USER

Log out and log back in for the changes to take effect.

Congratulations! You have successfully installed Docker on your system.

Next Steps

After installing Docker, you can start using Docker to create and manage containers. Refer to the Docker documentation for more information on using Docker and exploring its features.

Note: For Windows and macOS, Docker Desktop provides a graphical interface and additional tools for managing Docker containers. You can download Docker Desktop from the Docker website and follow the installation instructions specific to your operating system.
